* #### Переменные

    2 способа:

    ```php
    include 'file2.php';
    ```

* #### Массивы
    * **Что такое массив?**
    Массив (array) - упорядоченный набор данных, представляющий собой список однотипных элементов.
    Пример простого массива (элемент определяется индексом в последовательности):
    ```php
    <?php
    $colors = [
      'blue',
      'white',
      'yellow'
    ];
    ?>
    ```
    * **Что такое ассоциативный массив?**
    Массивы, где для обращения к элементам используются ключи, логически связанные со значениями.
    Пример ассоциативного массива:
    ```php
    <?php
    $person = [
      'age' => 21,
      'hair' => 'brown',
      'eyes' => 'green'
    ];
    ?>
    ```
    * **Как из массива/ассоциативного массива достать элемент?**
    Допустим, в file.php задан ассоциативный массив $colors (один из примеров выше). Чтобы вывести его значения в список в index.html нужно прописать:
    ```php
    ...
    <ul>
      <?php foreach ($colors as $color) :?>
          <li><?= $color; ?></li>
      <?php endforeach; ?>
    </ul>
    ...
    ```
    Если необходимо вывести не только значения, но и ключи:
    ```php
    ...
    <ul>
      <?php foreach ($person as $key => $feature) :?>
          <li><strong><?= $key; ?></strong>: <?= $feature; ?></li>
      <?php endforeach; ?>
    </ul>
    ...
    ```
* #### Функции
   * **Что такое функция?**
   Функция - блок кода, который имеет название и входные параметры для обработки и который может быть выполнен, когда это необходимо.
   Пример функции, принимающей два аргумента, и выводящей их:
   ```php
   <?php
   function functionName($arg1 = '', $arg2 = '') {
	    echo "arg1: $arg1\n";
	    echo "arg2: $arg2\n";
	}
  ?>
  ```
  Если вызвать функцию следующим образом:
  ```php
  <?php
  functionName('hello','world');
  ?>
  ```
  Выведется:
  ```
  arg1: hello arg2: world
  ```
  Существует множество встроенных функций, например:
  ```php
  <?php
  die() // выводит сообщение и прекращает выполнение текущего скрипта.
  ucwords() // преобразует в верхний регистр первый символ каждого слова в строке
  var_dump() // возвращает структурированную информацию об одном или более выражений, которая включает его тип и значение
  ...
  ?>
  ```
* #### Классы, ООП
  * **Что такое класс?**
  Класс - это шаблон кода, который используется для создания объектов. Класс определяется с помощью ключевого слова class после которого указывается произвольное имя класса.
  ```php
  <?php
    class copybook {
    // Тело класса
  }
  ?>
  ```
  * **Что такое метод класса? Что такое свойство класса?**
  В классе можно определить переменные. Переменные, которые определены в классе называются свойствами.
  ```php
  <?php
    class copybook {
      public $pages = 64;
      public $color = 'blue';
    }
  ?>
  ```
  Методы - это функции, которые определяются внутри класса, они позволяют объектам выполнять различные задачи. К методам объекта можно обращаться с помощь символов '->', указав объект и имя метода.
  ```php
  <?php
    class copybook {
      public $pages = 64;
      public $color = 'blue';

    function getpages() {
  echo $this->pages;
    }
  }
  ?>
  ```
  * **Зачем нужны публичные, приватные и защищенные методы/свойства?**
  Свойства и методы определяются с одним из ключевых слов protected, public или private, которые характеризуют управление доступом.
    * Public (общедоступные) свойства и методы. К ним можно получить доступ из любого контекста.
    * Protected (защищенные) свойствам и методы. К ним можно получить доступ либо из содержащего их класса, либо из его подкласса. Никакому внешнему коду доступ к ним не предоставляется.
    * Private (закрытые) свойства и методы. К ним можно получить доступ только из того класса, в котором они объявлены. Даже подклассы данного класса не имеют доступа к таким данным.

  * **Что такое инстанс (экземпляр класса)?**
  Экземпляр класса - это объект. Объект - это совокупность данных (свойств) и функций (методов) для их обработки.
  ```php
    <?php
    class copybook {
      public $pages = 64;
      public $color = 'blue';

    function getpages() {
  echo $this->pages;
    }
  }
  $copybook1 = new copybook();
  $copybook1->getpages(); //выведет "64"
    ?>
  ```
  * **Чем отличается статический метод класса от обычного? Как создать/вызвать статический метод?**
  Методы и свойства можно определять как статические данные класса. Доступ к их свойствам и методам возможен в контексте класса, а не объекта. Статические методы и свойства можно использовать, когда данные общие, о классе, а не о конкретном объекте. Определить статические данные класса можно с помощью ключевого слова static:
	```php
  <?php
    class copybook {
      static public $title = 1;

      static public function whatIsIt() {
        echo 'Тетрадь - письменный носитель информации, предмет для произведения записей, состоящий из листов бумаги, скреплённых различными способами.';
      }
    }
  ?>
  ```
  Статические методы сами не могут получать доступ ни к каким обычным свойствам класса, потому что такие свойства принадлежат объектам. Однако из статических методов можно обращаться к статическим свойствам.
  Обращение к классу/методу:
  ```php
  echo copybook::$title;
  copybook::whatIsIt();
  ```
  * **Как обычно именуют классы?**
  В имени класса может использоваться любое сочетание букв и цифр, но они не должны начинаться с цифры. Определение класса описывает, какие элементы будут содержаться в каждом новом экземпляре этого класса.
* #### PDO
  * **Что такое PDO? Зачем нужен этот класс?**
  PDO (PHP Data Objects) — расширение для PHP, предоставляющее простой интерфейс для доступа к различным базам данных.
  * **Как подключить PDO, чтобы начать с ним работать?**
  У сервера баз данных есть имя, также к нему могут подключаться пользователи, то есть для подключения должен быть использован логин и пароль.
  Общий пример того, как подключиться к базе данных:
  ```php
  <?php
  $pdo = new PDO("mysql:host=$host;dbname=$db", $user, $pass);
  ?>
  ```
  Более конкретный пример:
  ```php
  <?php
  $pdo = new PDO('mysql:host=127.0.0.1;dbname=mytodo', 'root', '1234');
  ?>
  ```
* #### Ошибки
  * **Что такое эксепшен (исключение)?**
  Исключение(exception) - это специальный объект, который является экземпляром встроенного класса Exception. Объекты типа Exception предназначены для хранения информации об ошибках и выдачи сообщений о них.
  * **Как поймать эксепшен и вывести пояснение, если что-то пойдет не так в коде?**
  Для этого необходимо поместить потенциально небезопасный код в блок try для перехвата исключения. Каждый try должен сопровождаться как минимум одним блоком catch.
  ```php
  <?php
  try {
    $pdo = new PDO('mysql:host=127.0.0.1;dbname=mytodo', 'root', '');
  } catch (PDOException $e) {
    die(e$->getMessage()); //выведет сообщение об ошибке
  }
  ?>
  ```
* #### Общие
  * **Что такое рефакторинг? Для чего он нужен?**
  «Рефакторинг» - это термин, который используется для описания процесса изменения структуры кода, не влияя на его функциональность.
  * **Что такое API?**
  API (application programming interface) – это интерфейс программирования - набор готовых классов, процедур, функций, структур и констант, предоставляемых приложением (библиотекой, сервисом)
  * **Что такое принцип единственной ответственности (one responsibility)?  **
  Single Responsibility Principle обозначает, что каждый объект должен иметь одну ответственность и эта ответственность должна быть полностью инкапсулирована в класс. Все его поведения должны быть направлены исключительно на обеспечение этой ответственности
